# Arcade (FB Alpha 2012)

### Description

Final Burn Alpha is an emulator of arcade games, including Capcom CPS-1 and CPS-2, SNK Neo-Geo, Toaplan, Cave and Sega System 16.

The 2012 version runs better on older hardware.
